## Tracks
- Tracks live under `conductor/tracks/`.
- Each track folder contains:
  - `index.md` (links)
  - `spec.md`
  - `plan.md`
  - `metadata.json`

## Templates

- Reusable conductor track templates live under `conductor/templates/`.
- The default high-rigor chained template for this repo is:
  - [nature_depth_cycle](./templates/nature_depth_cycle/README.md)
- That template is intended for one-aspect-at-a-time repository improvement with:
  - deep Ralph-style exploration;
  - Nature-level sophistication audit;
  - Conductor review and auto-remediation at each phase;
  - remote push, workflow monitoring, and Streamlit verification;
  - automatic creation of the next follow-on track;
  - reflection-driven improvement of the template itself.
