# Data Dictionary Template

For each dataset, include:

## Dataset name
## Purpose
## Source / custodian
## Legal basis / approvals
## Refresh cadence
## Unit of observation
## Time period covered
## Linkage keys (if any)
## Variables
| Variable | Type | Units | Allowed values | Missingness | Notes |
|---|---|---|---|---|---|

## Transformations
- Describe processing steps from raw → processed

## Quality checks
- Outlier rules
- Consistency rules
- Reconciliation totals

## Privacy / security
- De-identification
- Access controls
